#  Getting Started

## 1. Fork the Repository

Click the **Fork** button on GitHub to create your own copy.

---

## 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/hotflame-biogas.git
```

---

## 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

Example:

```bash
git checkout -b feature/improve-readme
```

---

## 4. Make Your Changes

Work on your improvements and ensure your code or documentation is clean and understandable.

---

## 5. Commit Your Changes

```bash
git commit -m "Add: meaningful description of changes"
```

---

## 6. Push to GitHub

```bash
git push origin feature/your-feature-name
```

---

## 7. Open a Pull Request

Submit a Pull Request describing:
- What you changed
- Why the change is important
- Any related issue
